# FAQ

## Q1: 服务器在美国，会不会时差导致错过数据？
不会。AQF 内部所有调度均以 `Asia/Shanghai`（北京时间）计算。

## Q2: Baostock 数据什么时候更新？
通常在 A 股收盘后晚上更新。AQF 从 20:00 开始每 5 分钟探测，直到检测到数据可用。

## Q3: Gmail 现在还能用密码登录吗？
一般不行。推荐开启两步验证并创建 **App Password**，然后把 App Password 填到 `.env`。

## Q4: 为什么建议 watchlist 10~80？
GA 评估成本大致与股票数量成正比。你可以先从少量开始，确认流水线稳定后再扩大。

## Q5: 这个策略能赚钱吗？
不做任何收益承诺。策略与参数由遗传算法在历史数据上拟合，存在过拟合风险。
请务必自行验证、加入更多约束、并在模拟盘长期观察。
